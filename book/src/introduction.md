# Star Atlas AI — OpenFang Agent OS

An open-source AI agent swarm for the Star Atlas Community and DAO, running on [OpenFang](https://www.openfang.sh/).

The swarm consists of two types of agents:

- **Hands** — autonomous agents that run on schedules, researching markets, brainstorming strategy, analyzing governance proposals, and curating knowledge without human intervention.
- **Agents** — interactive chat agents that community members spawn on-demand for gameplay advice, developer guidance, governance context, and lore exploration.

Both types draw on a shared knowledge architecture: a read-only **community vault** of curated Star Atlas knowledge and a read-write **knowledge vault** where hands deposit their output for other agents to consume.

**This is not financial advice.** All agent output is informational. Always do your own research (DYOR).
