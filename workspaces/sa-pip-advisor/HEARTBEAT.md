# Heartbeat Checklist

## Every Heartbeat (every 6h)
- [ ] Fetch current PIP listing from govern.staratlas.com
- [ ] Compare against previously reviewed PIPs in vaults/knowledge/pip-reviews/
- [ ] Analyze any new or updated PIPs — write reviews to vaults/knowledge/pip-reviews/
- [ ] Check community vault for governance context updates

## Daily
- [ ] Review existing PIP analyses for staleness (voting deadlines passed, status changed)

## Weekly
- [ ] Audit all active PIP reviews — update status fields
- [ ] Check for governance process changes or new PIP standards
