# Heartbeat Checklist

## Every Heartbeat (every 4h)
- [ ] Check community vault for updated context or new priorities
- [ ] Run market monitoring: Solana ecosystem, Web3 gaming news, Star Atlas updates
- [ ] Check ATLAS/POLIS token activity and notable on-chain events
- [ ] Write any new findings as research briefs to vaults/knowledge/inbox/
- [ ] Review unresolved leads and follow-up items from prior briefs in vaults/knowledge/inbox/

## Daily
- [ ] Check if any research briefs in inbox/ are older than 48h without being curated
- [ ] Update stale briefs with `status: stale-review-needed` and add a short note for sa-knowledge-keeper in the body

## Weekly
- [ ] Audit source freshness — flag any cited sources older than 90 days
- [ ] Review Star Atlas official channels for upcoming announcements or events
- [ ] Scan competing Web3 games for notable developments
