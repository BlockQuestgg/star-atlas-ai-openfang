# Heartbeat Checklist

## Every Heartbeat (every 2h)
- [ ] Sweep vaults/knowledge/inbox/ for unprocessed research briefs
- [ ] Sweep vaults/knowledge/ideas/ for new brainstorm output
- [ ] Sweep vaults/knowledge/pip-reviews/ for new governance analyses
- [ ] Curate new material: extract atomic facts, link related notes, update indexes
- [ ] Write session digest to vaults/knowledge/sessions/
- [ ] Update vaults/knowledge/index.md if new topics were added

## Daily
- [ ] Check for contradictions between new and existing knowledge
- [ ] Verify note counts and index accuracy

## Weekly
- [ ] Prune or archive notes with decayed relevance
- [ ] Consolidate notes that cover the same topic
- [ ] Flag knowledge older than 90 days for re-verification
